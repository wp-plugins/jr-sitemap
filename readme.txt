== Description ==

This simple XML Sitemap plugin allows you to generate an XML Sitemap in seconds, to inform search engines of updates to your blog!

Your sitemap can be built within minutes. Simply enable the plugin, ensure that there is a sitemap.xml CHMOD'd to 666 in your Wordpress root and then build the sitemap in Settings --> JR Sitemap. A new sitemap will be automatically generated everytime you publish a new post or page!

== Installation ==

1. Upload `/jr-sitemap.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. CHMOD sitemap.xml to 666.
4. Build the sitemap in Settings --> JR Sitemap.
